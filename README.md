# RYU QUEST — 大迫龍太 プロフィールサイト（レトロRPG風）

**公開URL: https://kaminendo-osako.github.io/ryu-quest/**

大迫龍太（Magician RYU × 株式会社かみねんど）のプロフィールを、
レトロRPG（コマンド選択式）のUIで楽しめるプロフィールサイト。

`index.html` をブラウザで開くだけで動きます（サーバー不要）。スマホ縦画面が基本、PCは中央レターボックス表示。

## 画面フロー

1. **タイトル画面** — ドット絵ポートレート＋「▶PUSH START」（タップで開始）
2. **オープニング** — タイプライター文字送り。時間帯であいさつが変化、再訪者には「◯かいめの ぼうけんだ」
3. **コマンドメニュー**
   - **つよさ** … マジシャンRYUのステータス画面（LVは1994年生まれから自動計算）＋クエストクリア記録＋実績
   - **しごと** … かみねんど／じゅもん「魔法の公式LINE」＋無料診断→公式LINE誘導
   - **どうぐ** … なんでも屋7品のアイテムメニュー（選ぶと説明が流れる）
   - **はなす** … お問い合わせ（公式LINE最優先／「てがみを かく」でメールフォーム）
   - **リンク** … Instagram・X・Web・公式LINE
4. **てがみ送信** → メールアプリ起動（osako@kaminendo.art宛）→ サンクス

## ゲーム演出

- ドラクエ風ウィンドウ（白枠二重線・黒背景）、DotGothic16 ピクセルフォント
- タイプライター文字送り＋▼点滅（タップ/Enter/Spaceで送り）
- ファミコン風効果音（WebAudio生成・右上「おと:ON/OFF」で切替）
- ドット絵ポートレート（canvasで写真をピクセル化）、星空、CRT走査線
- ボタン押下時のハプティクス（対応端末のみ）

## 編集ポイント

- 文章・選択肢：`index.html` 内の `SCENES` と `ITEMS`（JS部分）、パネルはHTML部分
- リンク類：JS冒頭の `LINK` オブジェクトにまとめてあります
- 効果音を消したいとき：`muted = false` を `true` に
